# Haoqun's Renovate Presets

Personal presets for [Renovate](https://github.com/renovatebot/renovate).

## Usage

Add this into `renovate.json`:

```json
{
  "extends": [
    "github>haoqunjiang/renovate-presets:npm.json5"
  ]
}
```
